区别

  1. 在函数前有一个关键字async，await关键字只能在使用async定义的函数中使用。任何一个async函数都会隐式返回一个promise，并且promise resolve的值就是 return 返回的值 (例子中是done)。
  2. 不能在函数开头使用await
